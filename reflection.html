Explain the concept of Responsive Design. What does the term mean?
Responsive design is also commonly kknown as 'Flexible Design'. The method of creating webages using flexible media queries, and graphics in the form of cascading style sheets.

It decreases the working of web designers to create websites for every single different device. Using responsive design, web designers can now create one site which changes it's layout.


How does responsive design relate to web technologies?
Online properties that are used on a device for watching is the process of responsive design. Th mobile platforms such as smartphones and tablets uses responsive design to scale it up to big screens.


What did you learn about building web apps for different devices through your research?
Every user struggle with storage concerns because as a typical mobile app weigh around 20MB (evontech.com, 2015). Due to the lack of storage capacity on mobile devices, it is impossible for a user to install every mobile app that looks to be useful for them. A web app is an ideal alternative in such scenario as it is only 1/20th the size of a mobile app and can be accessed quickly by simply typing a URL into a browser. A web app's URL can also be bookmarked for future usage, so users don't even need to remember it.

Moreover, businesses can save big bucks by using a web app since they don't have to build hefty, updated versions of mobile applications, and users don't have to upgrade their mobile apps as often. As a result, it wouldn't be an exaggeration to state that web apps benefit both owners and users.

Furthermore, it is easier to make a web app adaptable to smaller and larger screens and requires less effort, giving it a massive advantage over mobile apps.
